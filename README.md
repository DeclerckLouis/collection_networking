# Ansible Collection - packetflow.networking

A collection used to manage networking devices in the packetflow homelab.
Used to configure fortigate, ubiquiti and mikrotik devices.

It contains the following roles:  
    - dhcp_mgmt: to manage DHCP pools and reservations
    - firewall_mgmt: to manage firewall rules
    - vlan_mgmt: to manage VLANs

**Notice:** This collection is still in development and may contain bugs. Use it at your own risk.
